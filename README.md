# Browser Recorder Extension

一个 Chrome 浏览器扩展，用于录制和回放用户的浏览器操作行为。

## 功能特点

- 📝 **录制操作** - 记录点击、滚动、键盘输入等操作
- ▶️ **回放功能** - 重新执行录制的操作序列
- 💾 **导出/导入** - 支持 JSON 格式的导出和导入
- 🎨 **可视化指示** - 录制和回放时显示状态指示器

## 安装方法

### 开发者模式安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本文件夹

## 使用方法

### 开始录制

1. 点击扩展图标打开 popup
2. 点击「Start Recording」按钮
3. 在页面上进行操作
4. 点击「Stop Recording」停止录制

### 回放操作

1. 录制完成后，点击「Playback」按钮
2. 扩展会在当前标签页回放录制的操作

### 导出数据

1. 录制完成后，点击「Export JSON」
2. 选择保存位置，导出 JSON 文件

### 导入数据

1. 点击「Import」
2. 选择之前导出的 JSON 文件
3. 数据将被加载，可以进行回放

## 录制的数据类型

- **点击** - 记录点击的元素选择器和位置
- **键盘** - 记录按键事件（不记录具体输入内容）
- **滚动** - 记录滚动位置和方向
- **输入** - 记录输入框的类型和内容长度（不记录具体内容）
- **鼠标移动** - 记录鼠标位置

## 文件结构

```
browser-recorder-extension/
├── manifest.json          # 扩展配置
├── background.js          # 后台服务 worker
├── content_script.js      # 内容脚本（录制逻辑）
├── playback.js           # 回放脚本
├── popup.html            # 弹窗界面
├── popup.js              # 弹窗逻辑
├── icons/                # 图标文件夹
└── README.md             # 说明文档
```

## 技术栈

- Manifest V3
- Chrome Extension APIs
- 原生 JavaScript
- 无第三方依赖

## 注意事项

- 某些网站可能有内容安全策略限制
- 某些动态加载的元素可能无法正确回放
- 建议在测试环境中使用
